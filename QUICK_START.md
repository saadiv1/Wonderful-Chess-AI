# Quick Start

## Start the game

From this folder, run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

If you use VS Code, Live Server works too.

## Basic controls

- Click or tap a piece to select it
- Click a highlighted square to move
- Use `New Game` to restart
- Use `Undo` if your difficulty allows it
- Click a move in history to preview that position

## Before you play

You can choose:

- Side: White, Black, or Random
- Difficulty: Easy, Medium, Hard, or Insane
- Timer: no timer, bullet, blitz, or rapid presets

## If something looks broken

- Refresh the page
- Make sure the local server is running
- Open the browser console if you need to check for errors
