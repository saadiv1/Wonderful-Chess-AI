# Start Here

If you just want to run the game:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Useful files

- `README.md` - short project overview
- `QUICK_START.md` - how to play
- `INSTALLATION.md` - setup notes
- `config.js` - default settings

## Main code files

- `index.html`
- `styles.css`
- `main.js`
- `chess-engine.js`
- `ai.js`

That is enough to get started. If you are editing the game, `main.js` and `styles.css` are usually the first two places to look.
